# Travel Schedule Manager App

## Project Idea
A travel schedule manager app that helps users organize trip schedules, manage reservations, and calculate shared travel expenses.

## Features
- Organize travel schedules by date using a calendar
- Timeline view for schedules by time
- Reservation checklist
- Price split calculator
- Travel destination organizer

## Data
- Flight and transportation ticket dates
- Food and travel destinations
- Transportation options (Uber, bus, subway)
- Prices
- Tour costs
- Insurance fees
- Accommodation
